A simple alert box configureable by the admin. User get's a overlay div with the alert message once or more, just like you want it.

== Description ==

Well it is all what is says it is, an 'alert box', nothing more, nothing less. It displays a predefined message once. 
It uses the Lightbox-2 and the prototype window, both are included.

== Installation ==

1. Upload the whole 'wp-alertbox' directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure it in the configuration, and you are good to go.
